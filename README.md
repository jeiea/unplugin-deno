# unplugin-deno

deno unplugin. Simplified version of
[@bureaudouble/rolldown-deno-loader-plugin](https://jsr.io/@bureaudouble/rolldown-deno-loader-plugin).

## Rolldown usage

```ts
// rolldown.config.mts
import denoPlugin from "jsr:@jeiea/unplugin-deno/rolldown";

export default {
  plugins: [denoPlugin()],
};
```
